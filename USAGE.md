## üí¨ Messaging

### Starting a Conversation

1. **Navigate to Chats**
   - Click "Chats" in the sidebar
   - See your conversation list on the left

2. **Select a contact**
   - Click any conversation
   - Chat opens on the right

3. **Quick access from "Online Now"**
   - Check the right sidebar (desktop only)
   - Click any online user
   - Conversation opens automatically

### Sending Messages

**Text messages:**
- Type in the message input at the bottom
- Press Enter to send (Shift+Enter for new line)
- Your message appears in an orange bubble on the right

**Images:**

**Watch for replies:**
- After you send a message, wait a moment
- You'll see typing indicators (three pulsing dots)
- The other person replies automatically!
- Replies use contextual messages

### Managing Conversations

**Clear a conversation:**
- Click the ‚ãÆ menu in the chat header
- Select "Clear conversation"

**Mark as read:**
- Conversations mark as read automatically when viewed
- Unread badges update in real-time

## üîß Troubleshooting

### Common Issues

**Images not loading:**
- Check your internet connection
- Unsplash URLs require network access
- Fallback images will appear on error

**Theme not saving:**
- Check if localStorage is enabled
- Clear browser cache and try again
- Some browsers block localStorage in private mode

**Animations stuttering:**
- Close other browser tabs
- Update your browser to the latest version
- Disable browser extensions temporarily

**Mobile menu not opening:**
- Tap directly on the logo
- Ensure JavaScript is enabled
- Try refreshing the page
